# Initialize InvestKit Skill Research Project

## Objective

Initialize the InvestKit repository as a governed investment research AI capability package workspace.

## Scope

- Create project directory structure.
- Create product, Skill research, security, registry, template, and batch task documents.
- Create structured Schema and source registry.
- Preserve the existing Trellis block in `AGENTS.md` while adding InvestKit project instructions.
- Do not search the internet, download third-party Skills, install dependencies, execute third-party scripts, connect finance APIs, or implement trading features.

## Completion Criteria

- Required files and directories exist.
- Markdown path references point to real locations.
- Schema is valid JSON.
- CSV header is correct.
- Documents do not claim any Skill has already been collected or reviewed.
- No real trading implementation or API key is introduced.
- Git status is checked.
