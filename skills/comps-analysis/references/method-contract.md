# Comparable Companies Method Contract

Version: 1.0

## Peer Ledger

For every considered peer record `peer_id`, status, reason, business-model/mix/geography/scale/growth/margin/leverage/fiscal/accounting comparisons, source IDs, and warnings.

## Metric Observation

Record metric definition, enterprise/equity basis, numerator, denominator, period, estimate vintage, currency/conversion source/date, market-data as-of time, value or invalid state, and exclusion reason.

## Distribution

Use valid observations only and record sample size, median, quartiles, range, outlier policy, selected-rationale, implied target range, and EV-to-equity bridge.

## Guards

Never fill a denominator, silently drop a peer, use a hard-coded peer list as proof, or apply an automatic premium/discount.
