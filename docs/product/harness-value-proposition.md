# InvestKit Harness 的核心价值

直接使用单个 Skill 解决的是“调用一次工具”。InvestKit 解决的是“把一次研究变成可重复、可追溯、可恢复、可审计的投资研究过程”。

| 维度 | 直接使用 Skill | InvestKit Harness |
|---|---|---|
| 数据源选择 | 用户手工选择，容易漏源 | Catalog 按能力、审批、凭据和网络权限动态规划 |
| 身份一致性 | 每个 Skill 独立解析股票代码 | 先由交易所锁定身份，再拒绝 Provider 身份冲突 |
| 多源融合 | 返回结果彼此孤立 | 统一为九种 Provider 操作和一份不可变证据快照 |
| 事实可信度 | 结果通常缺少完整来源链 | 每项事实、估计、风险和结论绑定 `source_id` |
| 缺失数据 | 模型容易自行补全 | 未知项沿工作流传播，能力不足时明确跳过 |
| 分析框架 | 每次临时组织提示词 | 13 个版本化研究能力按固定依赖顺序运行 |
| 反方验证 | 通常只生成单边观点 | 独立熊案、来源验证和冲突检查是强制阶段 |
| 持久化 | 对话结束后难以复现 | 保存输入快照、计划、能力产物、日志和报告 |
| 恢复与诊断 | 失败后从头再来 | 支持 resume，Doctor 检查版本、来源、文件和凭据门控 |
| 安全边界 | Skill 自行读取 key、联网或遥测 | 环境变量、精确主机白名单、显式联网、响应限额、无遥测 |

## 当前已实现的融合路径

`股票代码 → SSE 官方身份/公告 → 广发 F10/相对估值 → 中金行情/四类财务/目标相关事件 → 身份与期间校验 → 不可变证据快照 → 13 能力工作流 → 市场量化背景/独立熊案/来源验证 → 持久化报告`

真实验收标的为飞科电器 `603868.SH`。Harness 无须指定同行即可自动加入广发目标公司 F10、PE/PB、行业均值及历史百分位，并以 `market-relative-valuation` 完成相对估值；中金路径提供价格、财务与目标相关事件。若分析师明确选择苏泊尔 `002032.SZ`，可再加入单一同行对比。DCF 与券商一致预期在当前获批数据不足时仍保持明确未知，不会从行业均值或历史增长中伪造。

## 使用方式

```bash
PYTHONPATH=src python3 -m investkit research \
  --symbol 603868.SH \
  --question '分析公司的基本面、财务质量、行业相对估值、市场表现、催化剂与主要风险。' \
  --allow-network
```

`--peer 002032.SZ` 是可选项，只应在分析师已经确认该公司具有经营和会计可比性时使用。

项目根目录的私有 `.env` 可由 CLI 安全加载：只接受登记的 Provider 变量、必须是普通文件且权限为 `600`，不会执行 shell 语句，也不会覆盖已经存在的进程环境变量。
