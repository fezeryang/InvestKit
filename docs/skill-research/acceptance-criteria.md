# Skill Research Acceptance Criteria

A candidate asset is research-complete only when all required items are finished or explicitly marked `unknown`.

## Required Criteria

- Source is traceable.
- Original files are preserved.
- Author or organization is recorded.
- License is identified or marked unknown.
- Actual capability is analyzed.
- Inputs, outputs, and dependencies are described.
- Required permissions are described.
- Security risks are described.
- Financial method issues are described.
- Duplicate relationships are analyzed.
- Target form is judged.
- Processing decision is explicit.
- Adaptation recommendation is explicit.
- Test recommendation is explicit.
- Structured record passes `registry/schema/skill-record.schema.json` validation.

## Completion Standard

The review must be honest and reproducible. Unknown information is acceptable only when it is marked as `unknown` and the reason is clear.
