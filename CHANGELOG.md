# Changelog

All notable InvestKit changes are documented here. Dates use Asia/Shanghai project
time. The project has not yet published a GitHub release.

## 0.3.0 — 2026-07-17

- Added provider-neutral real-company research bundles and immutable task resume.
- Added governed SSE, Guangfa, CICCWM, and reviewed Eastmoney provider boundaries.
- Added target-only A-share research from a stock code with automatic approved
  provider selection and optional analyst-selected peer comparison.
- Added point-in-time industry-relative valuation, forecast/consensus contracts,
  scenario DCF ranges when required evidence exists, and honest missing-data paths.
- Added recent return, realized-volatility, and maximum-drawdown market context.
- Added provider credential, network, identity, response-size, finite-number,
  provenance, migration, doctor, and artifact safety checks.
- Preserved the permanent prohibition on brokerage connectivity and trade execution.
