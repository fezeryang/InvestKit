# InvestKit Research Workspace

This directory is durable, user-owned InvestKit state. Each research run lives
under `research/<task-id>/` and contains its question, ordered plan, loaded specs,
installed Skills, normalized data, sources, assumptions, findings, risks, run log,
and report.

Tasks are designed to be readable without hidden chat history. Keep generated
task directories together when archiving or transferring research. The offline
demo is fictional, not live data, and is not a trading recommendation.
